.component()

Компоненты в Angular 1

Что такое компонент

  • элемент UI
  • может включать другие компоненты
  • максимально независимый от контекста использования

Задача

      
        
          <pokemon-list></pokemon-list>

Пример

      
pokemonApp.component('pokemonList', {
    controller: function PokemonListCtrl() {
        ...
    },
    templateUrl: './src/PokemonList/PokemonList.html'
});

Связывание данных

      
        
<pokemon-list-item pokemon="singlePokemon">
</pokemon-list-item>

Связывание данных


      
  pokemonApp.component('pokemonListItem', {
      bindings: {
          pokemon: '<'
      },
      templateUrl: './src/PokemonListItem/PokemonListItem.html'
  });

Restangular

https://github.com/mgonto/restangular

Подключение


npm install restangular --save
npm install lodash --save

Подключение

<script src="./restangular.js"></script> <script src="./lodash.js"></script>

Подключение


angular.module('myModule', ['restangular']);

Использование


Restangular.one('pokemon', 123).get().then(function(response) {
  $scope.pokemon = response
});

Встроенные функции


one(route, id)
all(route)
...
get([queryParams, headers])
put([queryParams, headers])
post(subElement, elementToPost, [queryParams, headers])
remove([queryParams, headers])
...

Использование


$scope.pokemon.put().then(function(successResult) {
  ...
});

Конфигурация


RestangularProvider.setBaseUrl('https://api.backendless.com/v1/data/');
